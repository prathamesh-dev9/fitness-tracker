# Health & Calorie Tracker

A premium, offline-first health and calorie tracker web application optimized for mobile devices. Built using vanilla web technologies, it provides a highly polished dark-themed user interface to log calories, macros, checklists, weight goals, and visual trends.

---

## ✨ Features

- **📱 Premium Mobile-First Design**: Modern, responsive layout with glassmorphic cards, harmonized accent colors (teal, blue, orange, pink), and smooth micro-animations.
- **🔥 Interactive Calorie Ring**: An SVG-powered daily calorie gauge showing total target progress and remaining calories.
- **🥩 Macro Nutrient Tracking**: Progress indicators for daily Protein, Carbs, and Fats.
- **🥑 Food Logging**:
  - Preloaded database of common Indian food items.
  - Search filter and instant calorie calculations.
  - Option to add custom items with custom names and calorie counts.
- **✅ Daily Checklist**: Multi-category task lists covering nutrition, hydration (water logs), exercise, and recovery sleep.
- **⚖️ Weight Goal Tracker**:
  - Tracks progress from a starting weight of 80 kg towards a 60 kg goal.
  - Historical logs tracking weight changes with auto-color-coded difference badges.
- **📊 Weekly Insights**:
  - Streaks and average daily calorie statistics.
  - Deficit computations and a dynamic weekly trend bar chart.
- **💾 Offline Backup & Import**: Complete local storage integration with export to `.json` files and backup import features.

---

## 🛠️ Technology Stack

- **Core Structure**: HTML5 (Semantic layout)
- **Styling**: Vanilla CSS3 (Custom variables, flexbox, CSS grid, keyframe animations)
- **Programming Logic**: Modern ES6+ JavaScript (State management, SVG attributes manipulation, local backup compilation)
- **Iconography**: Tabler Icons CDN
- **Fonts**: Google Fonts (Inter)

## 🚀 Getting Started

No installation or server setup is required. 

1. Clone or download this repository.
2. Double-click [index.html](file:///Users/prathamesh/repo/fitness-tracker/index.html) (or right-click and open with any modern web browser like Google Chrome, Safari, Firefox, or Microsoft Edge).
3. (Optional) Run a simple local HTTP server:
   ```bash
   npx serve .
   ```
